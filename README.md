# 🌤️ 오늘 날씨 웹앱

WeatherAPI.com을 활용한 실시간 날씨 정보 제공 웹 애플리케이션입니다.

## ✨ 주요 기능

- **실시간 날씨 정보**: 현재 날씨 상태를 실시간으로 확인할 수 있습니다
- **한국어/영어 검색 지원**: 한국어 도시명(서울, 부산 등)과 영어 도시명 모두 검색 가능
- **자동완성 기능**: 도시명 입력 시 자동완성 목록 제공
- **IP 기반 자동 위치 감지**: 페이지 로드 시 자동으로 현재 위치의 날씨 표시
- **상세 날씨 정보**: 온도, 체감온도, 습도, 풍속, 기압, 강수량, 자외선 지수, 가시거리, 구름량 등
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 UI

## 🚀 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다
2. 검색창에 도시 이름을 입력합니다 (예: 서울, 부산, 제주도, Seoul, London)
3. 자동완성 목록에서 원하는 도시를 선택하거나 검색 버튼을 클릭합니다
4. 해당 도시의 현재 날씨 정보가 표시됩니다

## 📁 파일 구조

```
weather/
├── index.html      # HTML 구조
├── style.css       # 스타일시트 (반응형 디자인)
├── script.js       # JavaScript 로직 (API 연동)
└── README.md       # 프로젝트 설명서
```

## 🛠️ 기술 스택

- **HTML5**: 웹 페이지 구조
- **CSS3**: 스타일링 및 반응형 디자인
- **JavaScript (ES6+)**: API 연동 및 동적 콘텐츠 처리
- **WeatherAPI.com**: 날씨 데이터 제공

## 📡 API 사용

이 프로젝트는 [WeatherAPI.com](https://www.weatherapi.com/)의 API를 사용합니다.

- **Current Weather API**: 실시간 날씨 정보 조회
- **Search/Autocomplete API**: 도시 검색 및 자동완성

### API 엔드포인트

- Current Weather: `https://api.weatherapi.com/v1/current.json`
- Search: `https://api.weatherapi.com/v1/search.json`

## 🎨 주요 기능 설명

### 1. 날씨 정보 표시
- 현재 온도 (섭씨/화씨)
- 날씨 상태 및 아이콘
- 체감 온도
- 습도, 풍속, 기압
- 강수량, 자외선 지수
- 가시거리, 구름량

### 2. 한국어 검색 지원
- 주요 한국 도시명을 영어로 자동 변환
- 검색 실패 시 자동완성 API를 통한 재시도
- 자동완성 결과 클릭 시 정확한 위치 정보 사용

### 3. 반응형 디자인
- 데스크톱: 최대 800px 너비의 카드 레이아웃
- 태블릿: 그리드 레이아웃 자동 조정
- 모바일: 단일 컬럼 레이아웃

## 🌍 지원하는 검색 형식

- 한국어 도시명: 서울, 부산, 제주도 등
- 영어 도시명: Seoul, Busan, London 등
- 위도/경도: `37.5665,126.9780` 형식
- IP 주소: 자동 위치 감지 (`auto:ip`)
